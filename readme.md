## scoping api (python3)

Задание в рамках обучения на курсе **Python Developer. Professional** (otus.ru)

#### Запуск скрипта

```
python3 api.py [--log logfile] [--port port]
```

| Param   | Decsription       |
| ------- | ----------------- |
| logfile | имя лог файла     |
| port    | порт http сервера |

#### Запуск тестов

```
python3 test.py
```
